# 📚 Ejercicios de Programación Web

Colección de 10 ejercicios prácticos de la materia **Programación Web** de la carrera de Ingeniería en Sistemas Computacionales. Cada ejercicio aborda conceptos fundamentales de desarrollo web front-end, desde la estructura básica de HTML hasta el uso de JavaScript para interactividad.

---

## Índice de Ejercicios

| # | Archivo | Tema | Descripción |
|---|---------|------|-------------|
| 1 | `ejercicio1.html` | Estructura HTML básica | Primer documento HTML: "Hola mundo" y párrafo introductorio. |
| 2 | `ejercicio2.html` | Formato de texto | Uso de etiquetas `<strong>`, `<em>` y `<br>` para dar formato al texto. |
| 3 | `ejercicio3.html` | Enlaces (hipervínculos) | Práctica con `<a>` usando `target="_blank"` y `target="_self"`, enlaces internos y externos. |
| 4 | `ejercicio4.html` | Imágenes | Inserción de imágenes con `<img>`, atributos `src`, `alt` y `width`. |
| 5 | `ejercicio5.html` | Integración HTML | Página tipo blog combinando texto, imágenes, enlaces y formato. |
| 6 | `ejercicio6.html` | HTML semántico | Uso de etiquetas semánticas: `<header>`, `<section>` y `<footer>`. |
| 7 | `ejercicio7.html` | Listas | Listas ordenadas (`<ol>`) y no ordenadas (`<ul>`) con listas anidadas. |
| 8 | `ejercicio8.html` | Tablas + CSS externo | Tabla de horario de clases con `rowspan`, `colspan` y hoja de estilos externa. |
| 9 | `ejercicio9.html` | Formularios + CSS externo | Formulario de contacto con campos de texto, email, teléfono y textarea. |
| 10 | `ejercicio10.html` | JavaScript | Convertidor de temperatura (°C → °F) con validación de entrada y lógica en JS. |

##  Tecnologías Utilizadas

- **HTML5** — Estructura y contenido de las páginas.
- **CSS3** — Estilos y diseño visual (inline y hojas externas).
- **JavaScript** — Interactividad y lógica del lado del cliente.

---

##  Cómo Ejecutar

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/tu-usuario/prograweb.git
   ```

2. **Abrir en el navegador:**
   - Abre `index.html` directamente en tu navegador, o
   - Si tienes un servidor local (como XAMPP), coloca la carpeta en `htdocs/` y accede a:
     ```
     http://localhost/prograweb/
     ```

##  Conceptos Practicados

- Estructura básica de un documento HTML (`<!DOCTYPE>`, `<html>`, `<head>`, `<body>`)
- Etiquetas de texto y formato (`<h1>` – `<h6>`, `<p>`, `<strong>`, `<em>`)
- Hipervínculos y navegación (`<a href>`, `target`)
- Multimedia (`<img>`)
- Etiquetas semánticas (`<header>`, `<section>`, `<footer>`)
- Listas ordenadas y desordenadas (`<ol>`, `<ul>`, `<li>`)
- Tablas con combinación de celdas (`<table>`, `rowspan`, `colspan`)
- Formularios (`<form>`, `<input>`, `<textarea>`, `<button>`)
- CSS externo con selectores de clase
- JavaScript: manipulación del DOM, validación de datos y eventos
